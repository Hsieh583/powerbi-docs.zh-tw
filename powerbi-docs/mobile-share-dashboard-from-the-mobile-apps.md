# <a name="share-a-dashboard-from-the-power-bi-mobile-apps"></a>從 Power BI mobile apps 共用儀表板
適用於︰

| ![iPhone](media/mobile-share-dashboard-from-the-mobile-apps/iphone-logo-50-px.png) | ![iPad](media/mobile-share-dashboard-from-the-mobile-apps/ipad-logo-50-px.png) | ![Android 手機](media/mobile-share-dashboard-from-the-mobile-apps/android-phone-logo-50-px.png) | ![Android 平板電腦](media/mobile-share-dashboard-from-the-mobile-apps/android-tablet-logo-50-px.png) | ![Windows 10](media/mobile-share-dashboard-from-the-mobile-apps/win-10-logo-50-px.png) |
|:--- |:--- |:--- |:--- |:--- |
| iPhone |iPad |Android 手機 |Android 平板電腦 |Windows 10 裝置 |

只要有 [Power BI Pro 授權](service-free-vs-pro.md)，您就可以從 Power BI 行動裝置應用程式來共用連結，以邀請同事檢視您的儀表板。 與您共用儀表板的人必須和您位在相同的電子郵件網域，同時也必須具有 Power BI Pro 授權，或內容必須位於[進階容量](service-premium.md)中。

從 Power BI for iOS 和 Android 行動裝置應用程式中，您也可以與任何人一起[標註並共用圖格、報告或視覺效果的快照集](mobile-annotate-and-share-a-tile-from-the-mobile-apps.md)。 

## <a name="share-a-dashboard-from-your-iphone"></a>從您的 iPhone 共用儀表板
1. 在儀表板底部的動作列中點選共用圖示 ![共用圖示](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-share-dashboard-icon.png).
   
   ![iPhone 共用圖示](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-dashboard-invite.png)
2. 點選 [邀請同事]。
3. 輸入名稱並以逗號分隔，再輸入要隨儀表板邀請一起發送的訊息。
4. 若要允許再次共用，請保持選取 [允許收件者共用此儀表板]  。
   
   再次共用可讓您的同事透過 Web 或行動裝置應用程式，將電子郵件邀請轉寄給組織中的其他人。
5. 點選右上角的 [傳送]  。
   
   您的同事會收到附有儀表板直接連結的電子郵件邀請函。 邀請有效期一個月。 當他們在瀏覽器或 Power BI 行動裝置應用程式中開啟邀請函時，它會新增至 Power BI 帳戶的 [與我共用] 區段。
   
   ![與我共用](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-shared-with-me-left-nav.png)
   
   進一步了解[與同事共用儀表板的相關注意事項](service-share-dashboards.md)。

### <a name="unshare-a-dashboard-from-your-iphone"></a>從您的 iPhone 取消共用儀表板
如果您是儀表板的擁有者，您只能取消共用儀表板。

1. 在儀表板底部的動作列中點選共用圖示 ![共用圖示](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-share-dashboard-icon.png).
2. 點選 [邀請同事]。
   
   隨即顯示您已與其共用此儀表板的同事清單與下列內容：
   
   * **可檢視**：可以檢視儀表板但不能共用它。
   * **可檢視和邀請**：可以檢視儀表板，也可與其他同事共用。
3. 點選 [編輯] 。
   
    ![編輯圖示](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-edit-invite-dashboard.png)
4. 若不共用，請點選名稱旁的紅色圓形並點選 [刪除] 。

## <a name="share-a-dashboard-from-your-ipad"></a>從您的 iPad 共用儀表板
1. 點選共用圖示 ![共用圖示](media/mobile-share-dashboard-from-the-mobile-apps/pbi_ipad_shareiconblk.png) 儀表板右上角的圖示。
2. 點選 [邀請同事] ，然後輸入電子郵件地址及隨儀表板邀請一起發送的訊息。
3. 若要允許再次共用，請保持選取 [允許收件者共用此儀表板]  。
   
   > [!NOTE]
   > 再次共用允許您的同事透過 Web 或 mobile apps，將電子郵件邀請轉寄給貴組織的其他人。
   > 
   > 
4. 點選右上角的 [傳送]  。
   
   您的同事會收到附有儀表板直接連結的電子郵件邀請函。 邀請有效期一個月。 當他們在瀏覽器或 Power BI 行動裝置應用程式中開啟邀請函時，它會新增至 Power BI 帳戶的 [與我共用] 區段。
   
   ![與我共用](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-shared-with-me-left-nav.png)
   
   進一步了解[與同事共用儀表板的相關注意事項](service-share-dashboards.md)。

### <a name="unshare-a-dashboard-from-your-ipad"></a>從您的 iPad 取消共用儀表板
如果您是儀表板的擁有者，您只能取消共用儀表板。

1. 點選共用圖示 ![共用圖示](media/mobile-share-dashboard-from-the-mobile-apps/pbi_ipad_shareiconblk.png) 在儀表板右上角。
   
   您會看到與其共用此儀表板的同事清單。 其名稱下有這些描述：
   
   * **暫止**：他們尚未檢視儀表板。
   * **可檢視**：可以檢視儀表板但不能共用它。
   * **可檢視和邀請**：可以檢視儀表板，也可與其他同事共用。
2. 點選 [編輯] 。
3. 若不共用，請點選名稱旁的紅色圓形並點選 [刪除] 。

## <a name="share-a-dashboard-from-your-android-device"></a>從您的 Android 裝置共用儀表板
1. 在儀表板首頁上，點選省略符號 (...)，然後點選 [邀請同事]。
   
   ![邀請同事](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-android-tablet-share-dashboard.png)
2. 或者點選儀表板右上角的邀請圖示 ![邀請圖示](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-android-invite-icon.png).

    如果您是儀表板的擁有者，您會看到與您共用此儀表板的同事清單及這些標註：

    -   **可檢視**：他們已檢視儀表板，但不能加以共用。

    -   **可檢視與邀請**：他們已檢視儀表板，並可與其他同事共用。

1. 請輸入電子郵件地址及隨儀表板邀請一起發送的訊息。 否則 Power BI 會傳送預設的訊息。
2. 若要允許再次共用，請保持選取 [允許收件者共用此儀表板]  。
   
   > [!NOTE]
   > 再次共用允許您的同事透過瀏覽器或 mobile apps，將電子郵件邀請轉寄給貴組織其他人員。
   > 
   > 
3. 點選傳送圖示 ![傳送圖示](media/mobile-share-dashboard-from-the-mobile-apps/pbi_andr_sendplane.png) 在右上角以傳送郵件。
   
   您的同事會收到附有儀表板直接連結的電子郵件邀請函。 邀請有效期一個月。 當他們在瀏覽器或 Power BI 行動裝置應用程式中開啟邀請函時，它會新增至 Power BI 帳戶的 [與我共用] 區段。
   
   ![與我共用](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-android-shared-with-me-left-nav.png)
   
   進一步了解[與同事共用儀表板的相關注意事項](service-share-dashboards.md)。

### <a name="unshare-a-dashboard-from-your-android-device"></a>從您的 Android 裝置取消共用儀表板
只有儀表板擁有者可以不共用儀表板。

1. 點選儀表板右上角的邀請圖示 ![邀請圖示](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-android-invite-icon.png). 
   
   您會看到與其共用此儀表板的同事清單。
2. 若要停止與同事共用儀表板，請點選名稱旁邊的 **X** \>**移除**.

## <a name="share-a-dashboard-from-your-windows-10-device"></a>從您的 Windows 10 裝置共用儀表板
1. 在儀表板中，點選邀請圖示 ![邀請圖示](media/mobile-share-dashboard-from-the-mobile-apps/pbi_andr_inviteicon.png).
   
   或者在儀表板首頁上，以滑鼠右鍵按一下或長按儀表板，然後點選 **邀請**。
   
   ![邀請](media/mobile-share-dashboard-from-the-mobile-apps/pbi_win10_sharedash.png)
   
   如果您是儀表板的擁有者，您會看到與您共用此儀表板的同事清單及這些標註：
   
   **唯讀**：他們已檢視儀表板，但無法加以共用。
   
   **讀取並再次共用**：他們已檢視儀表板，並可與其他同事共用。
2. 請輸入電子郵件地址及隨儀表板邀請一起發送的訊息。 否則 Power BI 會傳送預設的訊息。
   
   ![[邀請] 對話方塊](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-windows-10-share-dashboard.png)
3. 若要允許再次共用，請保持選取 [允許收件者共用此儀表板]  。
   
   > [!NOTE]
   > 再次共用允許您的同事透過瀏覽器或 mobile apps，與貴組織其他人員共用此儀表板。
   > 
   > 
4. 點選傳送圖示 ![傳送圖示](media/mobile-share-dashboard-from-the-mobile-apps/pbi_win10ph_sendicon.png).
   
   您的同事會收到附有儀表板直接連結的電子郵件邀請函。 邀請有效期一個月。 當他們在瀏覽器或 Power BI 行動裝置應用程式中開啟邀請函時，它會新增至 Power BI 帳戶的 [與我共用] 區段。
   
   ![與我共用](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-shared-with-me-left-nav.png)
   
   進一步了解[與同事共用儀表板的相關注意事項](service-share-dashboards.md)。
