# <a name="supported-browsers-for-power-bi"></a>支援 Power BI 的瀏覽器
Power BI 可在這些瀏覽器中執行：

* Microsoft Edge
* Internet Explorer 11
* Chrome 桌面最新版本
* Safari Mac 最新版本
* Firefox 桌面最新版本

所有適用的平台都支援這些瀏覽器。
