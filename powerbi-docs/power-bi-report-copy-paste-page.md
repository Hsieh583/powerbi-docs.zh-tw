# <a name="duplicate-a-report-page-in-power-bi"></a>在 Power BI 中複製報表頁面
1. 在 Power BI 的[編輯檢視](service-reading-view-and-editing-view.md)中開啟報表，如果您的報表尚未有視覺效果，請[建立視覺效果](power-bi-report-add-visualizations-i.md)。 
2. 從頂端控制列選取 **複製此頁面** 圖示。
   
   ![](media/power-bi-report-copy-paste-page/pbi_duplicate_new.png)
   
   您的新頁面已建立並成為使用中頁面。
3. (選擇性) [重新命名報表頁面](service-rename.md)。
