# <a name="reorder-pages-in-a-report-in-power-bi"></a>重新排列 Power BI 報表中的頁面
在 Power BI 中，報表是由一或多個頁面所組成。  每個頁面的底端都有一個索引標籤。  若要重新排列報表，只需要選取索引標籤，並將其拖曳到新位置。
